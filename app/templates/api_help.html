Документация API

[POST] http://egg.pythonanywhere.com/api/v1.0/ecg_data - служит для отправки данных экг за некий промежуток времени на сервер.

Параметры:

data - string - строка со значениями экг и метками времени (дельтой) в чередующимся порядке через пробел

email - string - строка с email пользователя

password - string - пароль пользователя

Пример:

{"data": "0 15 47 132 1560 3200", "email": "test@test.ru", "password": "123123"}

Можно использовать экстренный режим, указав в url параметр 'is_danger'=true

Пример: http://localhost:5000/api/v1.0/ecg_data?is_danger=true
